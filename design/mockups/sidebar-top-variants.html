Chewo — Sidebar-top layout variants

Six layouts for the top of the sidebar — the traffic-light strip, the Code/Notes switcher, the session-create actions, the secondary icon tools, and the search field. Only that region varies; below the search each frame just fades a session-list hint for context. Every frame is an honest 300px column on the cool-slate sidebar surface.

Base palette (cool slate) and accent — Azure hsl(210 70% 58%) — are placeholders under active exploration. This board is for comparing layouts only.
